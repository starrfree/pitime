# Welcome to PiTime
This website shows the index of the current time in [PI](https://en.wikipedia.org/wiki/Pi) (ùúã), [E](https://en.wikipedia.org/wiki/E_(mathematical_constant)) (e), [PHI](https://en.wikipedia.org/wiki/Golden_ratio) (ùúô, ùúë) and [GAMMA](https://en.wikipedia.org/wiki/Euler%27s_constant) (ùõæ) decimals. It is updated every second and shows the <ins>first</ins> instance of the concatenation of hours-minutes-seconds using the 24 hour format. 
_Example: 4:02:10 AM -> 16:02:10 -> 160210_.
